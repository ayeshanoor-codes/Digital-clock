# Digital Clock – Real-Time Display

A simple and visually appealing **Digital Clock Web App** built using **HTML, CSS, and JavaScript**.  
The clock updates every second using `setInterval()` and displays the current system time in a modern UI.

---

## Features
- Real-time time update (every 1 second)
- Gradient background design
- Glassy digital clock styling
- Smooth shadows and neon-like glow
- Fully responsive for mobile devices
- Clean and beginner-friendly JavaScript
---

## Technologies Used
- HTML5  
- CSS3  
- JavaScript (ES6)

---

## How to Run the Project

1. Download or clone the repository
2. Open the folder
3. Simply open **index.html** in any browser
4. No installation or server required
